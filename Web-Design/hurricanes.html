Where Do Hurricanes Come From?

This is an educational slider website that I designed and developed for an intermediate level interactive media college course in October 2021.

The general goal for this assignment was to make an educational slider site ending in a quiz. On the technical side, the intention was to find and integrate several different jQuery plugins to enhance the user's experience. For this assignment, I used four different plugins: (1) a plugin for the full-page slider, (2) a plugin for the quiz at the end of the slider, (3) a plugin for the hurricane spinner loader, and (4) a plugin for a modal that contains the references that I used.

Each slide on the site has a full-page gif as a background in order to make the site feel more fun and eye-catching for the user, and there is use of diagrams, tables, and images where possible to go along with the information. The slider is packed full of information about how hurricanes form and is followed by a quiz that tests the user's knowledge. Pictured below is an example of one of the slides and the quiz.
